# Политика безопасности

## Поддерживаемая версия

Поддерживается только содержимое ветки `main`.

## Сообщение об уязвимости

Не публикуйте уязвимости, секреты или приватные VPN-конфигурации в открытых
issue. Используйте GitHub Private Vulnerability Reporting в разделе
**Security → Advisories → Report a vulnerability**, если он доступен.

Если проблема относится к самому приложению Anywhere, сообщите о ней напрямую
в https://github.com/NodePassProject/Anywhere.

## Область проекта

Репозиторий распространяет статические routing-списки и инструменты их
генерации. Он не принимает, не хранит и не обрабатывает VPN-ключи,
пользовательские подписки или сетевой трафик.
