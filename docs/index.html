01 / УСТАНОВКА

Три действия.
Несколько точных правил.

Anywhere ограничивает один набор десятью тысячами правил, поэтому ссылки добавляются отдельно.

  1. 01

    Откройте Anywhere

    Перейдите в Settings → Routing Rules → Subscribe.

  2. 02

    Скопируйте ссылки

    Добавьте все ссылки одного выбранного профиля по очереди.

  3. 03

    Назначьте действия

    Укажите DIRECT, REJECT или ваш VPN согласно карточкам.

02 / ПРОФИЛИ
PROFILE / DEFAULT

Сбалансированный режим

RU/BY и доверенные сервисы напрямую. YouTube, Telegram, GitHub и другие выбранные сервисы через VPN. Реклама и телеметрия блокируются.

Важно

Не устанавливайте DEFAULT и WHITELIST одновременно.

03 / ОБНОВЛЕНИЯ

Ссылка остаётся.
Правила меняются.

Откройте подписанный набор в Routing Rules и нажмите Update. Назначенное действие сохранится.
